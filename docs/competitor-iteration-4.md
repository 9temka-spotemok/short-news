## Итерация 4 — Расширенная аналитика и экспорт

### Цель
- Предоставить глубинный анализ конкурентной активности, трендов и изменений, а также расширенные возможности экспорта и повторного использования отчётов.

### Основные задачи
- **Метрики и аналитика**: построить тренды по тональности и объёму новостей, сравнение тарифных уровней, оценку скорости инноваций; рассчитать комбинированный `impact_score`, объединяющий новости, изменения цен и новые функции; создать граф связей «событие → последствие».
- **Knowledge Graph**: связать сущности (новость, тариф, функция, команда, продукт) для выявления причинно-следственных связей и сценариев.
- **Фронтенд**: добавить вкладки «Постоянные метрики» и «Актуальные сигналы», таймлайны изменений, сценарии сравнения (A/B), сохранение пользовательских отчётов (пресетов фильтров и компаний), отображение `impact_score`.
- **Экспорт**: обновить JSON/PDF/CSV выгрузки, включить новые метрики, историю изменений, настройки уведомлений; предусмотреть версии экспортов и опцию shareable link.
- **API**: ввести версионирование (`v2`) и фича-флаги для постепенного развертывания; документировать схемы `impact_score`, пресетов и экспортов.
- **Риски и производительность**: оптимизировать расчёты метрик (кеширование, фоновое обновление), добавить нагрузочные тесты, следить за деградацией NLP-моделей.
- **Документация**: обновить `README` (карта сервисов, инструкции по экспорту, описания ключевых UI-блоков), подготовить гайд для аналитиков по работе с расширенными метриками.
- **Тестирование**: e2e сценарии экспорта, визуальные тесты графиков/таймлайнов, интеграционные тесты версионированных API, проверка сохранения/загрузки пользовательских отчётов.

### Критерии готовности
- Расширенные метрики и графы отображаются в UI и доступны через API `v2`.
- Пользовательские отчёты сохраняются и повторно применяются.
- Экспорты содержат полный набор новых данных в каждом формате.
- Производительность расчётов находится в допустимых пределах, есть фич-флаги для контроля релиза.
- Документация отражает новые метрики, экспорт и ответственность компонентов.
